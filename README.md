# Personal_API_DIO
API REST com Spring Boot para cadastro e gerenciamento de pessoas de uma organização

```
## Projeto DIO  API REST com Spring Boot para cadastro e gerenciamento de pessoas de uma organização, até o Deploy na nuvem (Heroku).
Para pra ticar e ver os principais conceitos do estilo arquitetural REST envolvidos durante o desenvolvimento do projeto.

## O projeto foi desenvolvido em 
-- java 11 
-- Maven
-- SpringBoot 2.5.2
-- Packaging Jar
-- IntelliJ Idea

## site do aplicação criada 

-- https://start.spring.io/

``` 
```
## O projeto serve como referencia um controle de pessoas em uma organização,, que também é possivel uma atualização do mesmo.
Algumas atribuições podem ser criadas, com os packages criados e classes eé possivel notar do que se trata o objetivo do projeto.
Não somente cadastro mas exclusão e alteração de dados ja existentes.

## O projeto está em cloud e toda atualização vai ser notificada na nuvem pois o projeto está em seu inicio e posteriormente terá atualização
O programa Postamn também foi usado assim como o map Struct
```

```
Antes de rodar o projeto é necessário fazer as configuraões de acordo com o java que cada máquina usa.
```
